# **App Name**: MailZen Dashboard

## Core Features:

- Gmail Inbox Display: Display user's Gmail inbox messages, fetched directly from the Gmail API.
- Message Reply: Allow users to reply to emails directly within the dashboard using Gmail API.
- Mark as Read/Unread: Enable users to mark emails as read or unread, syncing status with Gmail.
- Insert Message Template: Allow users to insert pre-defined message templates into email replies. Powered by AI tool. AI suggests an appropriate template or creates one using generative AI based on the content of the last few messages exchanged with that addressee.
- Send Message: User can create a message and send it to the intended email.
- Search Functionality: Implement a search bar to find emails by sender, recipient, subject, or content.
- Email Categorization: Automatically categorize emails (e.g., primary, social, promotions) using keywords in the subject.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to promote a sense of calm and trustworthiness.
- Background color: Light grey (#F0F0F0) to maintain a neutral and bright theme.
- Accent color: Subtle orange (#FFB347) to draw attention to key actions and notifications.
- Body and headline font: 'Inter', a grotesque-style sans-serif, providing a modern and neutral aesthetic suitable for both headings and body text.
- Use minimalistic icons from a library like Phosphor Icons to maintain a clean and modern look.
- Employ a clean, card-based layout for email display. Use Shadcn UI components for consistency.
- Subtle animations for loading and transitions using libraries within Shadcn UI to add polish.
